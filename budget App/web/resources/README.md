Place production source assets here before running:

```bash
cd "budget App/web"
npm run cap:assets
```

Required source files:
- `resources/icon-only.png` (512x512 PNG)
- `resources/icon-foreground.png` (512x512 PNG)
- `resources/icon-background.png` (512x512 PNG)
- `resources/splash.png` (2732x2732 PNG recommended)

The generated assets will be written to both `android/` and `ios/` projects.
